# 自动追剧

自动追剧用于快速创建剧集追更任务。系统会根据剧名检索 TMDB 信息，再通过 CloudSaver 搜索天翼云盘分享资源，并按所选模式创建普通转存或懒转存任务。

---

## 1. 功能概述

自动追剧会串联以下能力：

- **TMDB 识别**：根据剧集名称和年份匹配剧集信息、总集数、评分与状态。
- **CloudSaver 搜索**：自动搜索公开天翼云盘资源，并按标题、年份等信息给出候选资源。
- **快速建任务**：使用系统预设的默认账号和默认保存目录创建任务。
- **媒体联动**：任务创建后可继续使用刮削、自动整理、STRM 和通知能力。

当前 Web 页主要提供“快速创建”入口，追更任务的后续执行和状态仍在任务系统中管理。

---

## 2. 前置配置

使用前请先完成两类配置。

### 2.1 自动追剧默认配置

进入 **系统** 页，在 **自动追剧默认配置** 中设置：

| 配置项 | 说明 |
| :--- | :--- |
| 默认追剧账号 | 自动追剧创建任务时使用的天翼云盘账号 |
| 默认保存目录 | 资源最终转存或懒转存触发后的保存目录 |

如果未配置，自动追剧页会提示：`请先到“系统”页配置自动追剧默认账号和默认保存目录。`

### 2.2 媒体与搜索配置

建议进入 **媒体** 页检查以下配置：

| 配置项 | 用途 |
| :--- | :--- |
| TMDB API Key | 提高剧集匹配和总集数识别准确度 |
| CloudSaver 设置 | 用于搜索可用分享资源 |
| STRM 设置 | 影响普通任务或懒转存任务的 STRM 生成 |
| AI 辅助重命名 / 媒体库分类命名 | 配合自动整理使用 |

## 3. Web 端使用

### 3.1 添加追剧

1. 进入 **自动追剧** 页。
2. 点击 **添加追剧**。
3. 填写 **剧集名称**。
4. 可选填写 **年份 (可选)**，用于区分同名剧集。
5. 选择模式：
   - **懒转存 (生成STRM)**：优先生成可播放的懒转存 STRM，播放时再触发转存。
   - **自动转存 (下载文件)**：立即创建并执行普通转存任务。
6. 点击 **开始追剧**。

### 3.2 手动选择资源

如果希望确认系统选中的资源：

1. 在添加追剧弹窗中勾选 **手动选择资源**。
2. 点击 **搜索资源**。
3. 在 **选择资源** 页面查看候选结果。
4. 参考 **匹配度**、资源标题、分享链接和 TMDB 信息选择资源。
5. 点击 **使用该资源创建任务**。

手动选择资源不是手动粘贴分享链接，而是从系统搜索到的候选资源中选择一个更合适的结果。

---

## 4. 两种模式说明

| 模式 | 适合场景 | 行为 |
| :--- | :--- | :--- |
| 懒转存 (生成STRM) | 想先入库，播放时再占用网盘空间 | 创建懒 STRM 相关任务，播放时触发转存并换取直链 |
| 自动转存 (下载文件) | 希望资源立即进入网盘 | 创建普通转存任务并立即执行 |

懒转存模式要求默认账号已经配置本地 STRM 目录，否则系统会提示无法执行懒转存。

---

## 5. 创建后的默认行为

自动追剧创建的任务默认会带有以下任务信息：

- 任务分组：`自动追剧`
- 备注：`自动追剧`
- 普通模式：启用刮削、自动整理，不启用懒 STRM
- 懒转存模式：启用懒 STRM、自动整理，不启用任务刮削

实际任务执行结果仍以任务列表和日志为准。

---

## 6. Telegram Bot 使用

Telegram Bot 也支持创建追剧任务：

```text
/series 剧名 年份
```

示例：

```text
/series 庆余年 2024
```

创建懒转存追剧：

```text
/lazy_series 剧名 年份
```

Bot 追剧同样会使用 **系统** 页中的自动追剧默认账号和默认保存目录。

## 7. 常见问题

### 搜索不到资源怎么办？

- 确认 **媒体** 页中的 CloudSaver 配置可用。
- 使用更准确的官方剧名。
- 添加年份，例如 `庆余年 2024`。
- 换用常见别名或原始标题再次搜索。

### 选到的资源不对怎么办？

创建时勾选 **手动选择资源**，在候选列表中按标题、年份、匹配度和分享链接选择。

### 懒转存失败怎么办？

检查默认账号是否配置了本地 STRM 目录，并确认 **媒体** 页的 STRM 设置已按需要启用。

---

## 使用建议

- 同名剧集尽量填写年份。
- 首次使用建议勾选 **手动选择资源**，确认搜索质量后再使用自动选择。
- 自动追剧依赖 CloudSaver 和 TMDB，搜索失败时优先检查这两个配置。
