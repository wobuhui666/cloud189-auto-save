# 媒体设置

**媒体** 页集中管理 AI 重命名、STRM、CAS 秒传、Emby、CloudSaver、Alist、TMDB 和媒体库分类等配置。

---

## 1. AI 辅助重命名

AI 辅助重命名用于整理器、任务重命名和媒体识别。

| 配置项 | 说明 |
| :--- | :--- |
| 启用 / 停用 AI | 控制 AI 功能是否生效 |
| API 地址 | OpenAI 兼容接口地址 |
| API Key | 接口密钥 |
| 模型名称 | 使用的模型名称 |
| 剧集命名模板 | 剧集整理后的文件名模板 |
| 电影命名模板 | 电影整理后的文件名模板 |
| AI API 流控 | 限制 AI 请求频率 |

模板示例：

```text
电视剧：{name} - {se}{ext}
电影：{name} ({year}){ext}
```

常用变量包括 `{name}`、`{season}`、`{episode}`、`{se}`、`{year}`、`{ext}`，实际支持情况以当前版本为准。

---

## 2. STRM 设置

| 配置项 | 说明 |
| :--- | :--- |
| 启用 STRM 生成 | 普通任务完成后生成 STRM |
| 普通任务使用系统中转 | 普通任务的 STRM 使用本项目代理地址，而不是直接写入临时直链 |

更完整的配置方式请参考 [[StrmGuide]]。

---

## 3. 秒传设置

| 配置项 | 说明 |
| :--- | :--- |
| 启用家庭中转 | 秒传时通过家庭云中转，规避个人云风控 |
| 优先使用家庭中转 | 默认先尝试家庭云，失败后回退个人云 |
| 恢复后删除 CAS 文件 | 秒传恢复成功后删除 `.cas` 存根文件 |
| 生成后删除源文件 | 生成 CAS 后自动删除源文件 |

页面提示中说明：家庭中转适用于个人云被风控或黑名单的情况，秒传恢复和懒 STRM 都会使用这些设置。

---

## 4. Emby 设置

| 配置项 | 说明 |
| :--- | :--- |
| Emby 地址 | Emby/Jellyfin 服务地址 |
| API Key | 媒体服务器 API Key |
| 启用 Emby 入库通知 | 任务完成后通知媒体服务器刷新 |
| 启用 Emby 反代播放 | 使用本项目反代播放地址 |
| 代理端口 | 独立代理端口，默认常用 `8097` |
| 启用下一集预热 | 监听播放会话并预热下一集 |
| Sessions 轮询间隔(ms) | 检查播放会话的间隔 |
| 预热去重时长(ms) | 避免短时间重复预热同一内容 |

如果使用 Docker，记得映射代理端口，例如 `-p 8097:8097`。

---

## 5. CloudSaver 设置

CloudSaver 用于资源搜索、自动追剧候选资源检索，以及任务失败后的自动换源。

| 配置项 | 说明 |
| :--- | :--- |
| 服务地址 | CloudSaver 服务地址 |
| 用户名 | CloudSaver 登录用户名 |
| 密码 | CloudSaver 登录密码 |

推荐使用项目 README 中提到的修改版 CloudSaver，以获得更适配的资源搜索能力。

---

## 6. Alist 设置

| 配置项 | 说明 |
| :--- | :--- |
| Alist 地址 | Alist 服务地址 |
| Token | Alist 管理员或可用 API Token |

Alist 可用于目录访问、直链获取和部分 STRM 联动场景。请确保本项目所在环境能访问 Alist 地址。

---

## 7. TMDB 刮削

| 配置项 | 说明 |
| :--- | :--- |
| TMDB API Key | 用于搜索剧集/电影、获取标题、年份、海报和集数等信息 |

TMDB 配置会影响自动追剧、任务刮削、整理器识别等功能的准确度。

---

## 8. 媒体库分类命名

用于整理器按媒体类型归档。

| 分类 | 默认用途 |
| :--- | :--- |
| 电视剧 | 剧集资源 |
| 动漫 | 动漫资源 |
| 电影 | 电影资源 |
| 综艺 | 综艺节目 |
| 纪录片 | 纪录片资源 |

可以按自己的媒体库目录命名习惯修改，例如把“电视剧”改为“剧集”。

---

## 9. 正则预设管理

媒体页提供 **正则预设管理**，用于维护任务创建时可复用的过滤和重命名规则。

预设字段包括：

| 字段 | 说明 |
| :--- | :--- |
| 名称 | 预设名称 |
| 描述 | 规则用途说明 |
| 源正则 | 匹配原文件名的正则 |
| 目标正则 | 替换后的命名规则 |
| 筛选模式 (正则) | 用于包含或排除文件的匹配规则 |
| 运算符 | 与匹配值配合用于条件过滤 |
| 匹配值 | 条件过滤值 |

更多正则示例请参考 [[Regex]]。

---

## 10. 保存配置

修改媒体页配置后：

- 点击 **保存媒体设置** 保存。
- 点击 **放弃修改** 恢复到保存前状态。

保存 CloudSaver 配置后，系统会清理旧的 CloudSaver 登录缓存；保存 Emby/CAS 等配置后，相关服务会按需要重新加载。

---

## 配置建议

- 自动追剧用户优先配置 TMDB 和 CloudSaver。
- 媒体服务器用户优先配置 STRM 设置、Emby 设置和代理端口。
- 文件名混乱时再开启 AI，简单过滤优先使用正则预设。
