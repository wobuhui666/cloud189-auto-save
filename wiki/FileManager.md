# 文件管理

**文件** 页用于在 Web 控制台中直接浏览和管理天翼云盘目录，无需离开本系统。

---

## 1. 功能概述

- 按账号切换个人云目录树（从「全部文件」根 `-11` 开始）。
- 面包屑导航进入子目录。
- 关键字过滤当前目录列表。
- 多选后批量移动、删除。
- 新建目录、重命名、复制文件 ID / 直链。

接口前缀：`/api/file-manager/*`，详见 [[API]]。

---

## 2. 使用步骤

1. 进入 **文件** 页。
2. 在顶部选择已登录的天翼账号（扫码/Cookie 临时账号名以 `n_` 开头的可能被过滤）。
3. 点击文件夹进入下一级；点面包屑可回退。
4. 需要时勾选多个文件/目录，使用 **移动选中** 或 **删除选中**。
5. 单文件菜单支持：复制 ID、复制直链、重命名、删除。

### 新建目录

点击 **新建目录**，输入名称后在当前目录创建文件夹。

### 移动

选择目标目录（须为**同一账号**下的路径，不能跨账号移动）。

### 直链

通过 `GET /api/file-manager/download-link` 获取临时下载地址，便于调试播放或外链。

---

## 3. 与任务系统的关系

- 文件管理操作直接改云盘，不会自动改写任务缓存；若移动了任务目标目录中的文件，可在任务页对相关任务执行 **清理缓存**。
- 整理器、转存任务仍通过任务系统改名/归档；文件页适合临时维护与排查。

---

## 4. 注意

- 删除为云盘侧删除，请谨慎操作。
- 家庭云账号的目录行为取决于账号类型与 Family 配置。
- 频繁列目录会请求天翼接口，注意账号风控与代理设置。
