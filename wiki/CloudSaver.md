# CloudSaver 搜索

CloudSaver 用于搜索公开的天翼云盘分享资源，并把结果快速带入任务创建流程。它既可以在 Web 中使用，也可以通过 Telegram Bot 搜索和转存。

---

## 1. 功能概述

CloudSaver 主要提供：

- **资源搜索**：按关键字搜索公开分享资源。
- **快速建任务**：搜索结果可直接带入创建任务弹窗。
- **自动追剧候选资源**：自动追剧会用 CloudSaver 搜索候选剧集资源。
- **Bot 搜索模式**：在 Telegram 中通过编号选择资源后直接转存。

CloudSaver 只负责搜索和返回分享链接，资源最终是否可用仍取决于分享本身和账号状态。

---

## 2. 前置配置

进入 **媒体** 页，在 **CloudSaver 设置** 中填写：

| 配置项 | 说明 |
| :--- | :--- |
| 服务地址 | CloudSaver 服务地址 |
| 用户名 | 登录用户名 |
| 密码 | 登录密码 |

未配置时，Bot 会提示：`未开启CloudSaver，请先在网页端配置CloudSaver`。

---

## 3. Web 端使用

CloudSaver 不是左侧导航页，而是一个全局浮动入口。

使用方式：

1. 打开右下角浮动按钮。
2. 点击 **CloudSaver**。
3. 在弹窗 **CloudSaver 资源搜索** 中输入关键字。
4. 点击 **搜索**。
5. 在结果列表中点击右侧的一键转存按钮。
6. 系统会自动打开创建任务弹窗，并带入：
   - `shareLink`
   - `accessCode`
   - `taskName`

搜索结果会显示标题、大小和日期；部分资源可能带访问码。

---

## 4. 与自动追剧的关系

自动追剧创建流程会先根据 TMDB 信息生成候选标题，再调用 CloudSaver 搜索资源。

典型流程：

```text
输入剧名
  ↓
TMDB 匹配剧集信息
  ↓
CloudSaver 搜索候选资源
  ↓
自动选择最佳资源或手动选择资源
  ↓
创建普通任务或懒转存任务
```

如果自动追剧搜索不到候选资源，优先检查 CloudSaver 配置是否可用。

---

## 5. Telegram Bot 使用

在 Telegram 中发送：

```text
/search_cs
```

进入搜索模式后：

1. 输入关键字，例如 `庆余年`。
2. Bot 返回编号列表。
3. 发送编号，例如 `1`。
4. Bot 会把对应分享链接带入转存流程。

搜索模式特点：

- 需要先用 `/accounts` 选择账号。
- 3 分钟无操作会自动退出搜索模式。
- 输入 `/cancel` 可以主动退出。
- 如果在搜索模式中直接发送分享链接，Bot 会先退出搜索模式，再按分享链接创建任务。

---

## 6. API 接口

所有接口都需要带上系统 API Key：

```http
x-api-key: YOUR_API_KEY
```

### 搜索资源

```http
GET /api/cloudsaver/search?keyword=关键词
```

返回结果中会包含资源标题和可用的 `cloud.189.cn` 分享链接。

---

## 使用建议

- 搜索电影时建议加年份，例如 `阿凡达 2009`。
- 搜索剧集时建议用官方中文名；如果结果不佳，可以改用原始标题再试。
- 搜索结果过多时，优先结合年份、分辨率或制作组关键词缩小范围。
