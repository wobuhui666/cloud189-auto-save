# 订阅

订阅功能用于管理外部资源集合。用户可以通过 UUID 或订阅主页链接导入资源列表，校验资源可用性，并按需创建转存任务或自动创建任务。

---

## 1. 功能概述

订阅系统支持：

- **UUID / 订阅主页链接导入**：添加外部订阅源。
- **远程资源选择**：只选择订阅内需要的资源。
- **资源校验**：检查分享链接是否可用，并记录可用账号。
- **自动创建任务**：按默认账号、目录和任务配置批量创建任务。
- **资源管理**：浏览、转存、删除订阅资源。

---

## 2. 添加订阅

进入 **订阅** 页，点击 **添加订阅**。

基础字段：

| 字段 | 说明 |
| :--- | :--- |
| UUID / 订阅主页链接 | 资源分发者提供的 UUID 或订阅页面地址 |
| 名称 (可选) | 自定义订阅名称，留空时系统按订阅信息生成 |
| 备注 | 给自己看的说明 |
| 启用此订阅 | 控制订阅是否参与后续刷新和管理 |

填写 UUID 后，系统会给出预估信息，例如是否已存在同 UUID 订阅、远程资源数量、默认账号是否可用等。

---

## 3. 选择订阅内资源

如果订阅源包含很多资源，可以点击 **选择订阅链接** 只导入部分资源。

远程选择窗口支持：

| 功能 | 说明 |
| :--- | :--- |
| 按标题搜索订阅里的链接 | 按资源标题过滤 |
| 选择 | 勾选需要导入的资源 |
| 资源 | 查看资源标题 |
| 类型 | 区分文件或文件夹 |
| 分享码 | 查看对应分享标识 |
| 更新时间 | 判断资源新旧 |
| 打开详情链接 | 打开资源详情页 |

也可以使用 **清空选择** 恢复为不限制导入范围。

---

## 4. 自动创建任务

在添加或编辑订阅时，可以开启 **自动创建任务**。

自动任务配置字段包括：

| 字段 | 说明 |
| :--- | :--- |
| 默认账号 | 创建任务使用的账号 |
| 默认保存目录 | 创建任务的目标目录 |
| 任务分组 | 自动创建任务所属分组 |
| 任务备注 | 写入任务备注 |
| 启用刮削 | 创建任务时启用 TMDB 刮削 |
| 懒 STRM | 创建懒转存相关任务 |
| 自动整理 | 创建任务时启用整理器 |
| 定时任务 | 为创建出的任务启用 Cron 检查 |
| Cron 表达式 | 定时任务执行周期 |

系统会显示 **任务预估**，包括订阅资源数、预计任务数、失败预估数和建议说明。默认账号和目录通常来自 **系统** 页的自动追剧默认配置，也可以在订阅自动任务配置中调整。

---

## 5. 管理订阅

订阅列表字段：

| 字段 | 说明 |
| :--- | :--- |
| 名称 | 订阅名称 |
| UUID | 订阅唯一标识 |
| 资源数 | 已导入资源数量 |
| 状态 | 最近刷新结果 |
| 账号覆盖 | 可用账号覆盖情况 |
| 最后检查 | 最近一次刷新或校验时间 |

常用操作：

- **查看资源**：打开该订阅的资源列表。
- **立即校验**：刷新订阅并校验资源状态。
- **启用 / 停用**：控制订阅是否继续使用。
- **编辑**：修改订阅信息和自动创建任务配置。
- **删除**：删除订阅记录。

---

## 6. 管理订阅资源

在资源列表中可以看到：

| 字段 | 说明 |
| :--- | :--- |
| 资源 | 资源标题和分享链接 |
| 类型 | 文件或文件夹 |
| 状态 | 可用、失效或未知 |
| 可用账号 | 当前能访问该资源的账号 |
| 最后校验 | 最近一次校验时间 |
| 更新于 | 资源更新时间 |

资源操作包括：

- **浏览内容**：查看资源目录内容。
- **转存**：把该资源带入创建任务流程。
- **删除**：从订阅资源列表中移除。
- **全部校验**：批量刷新资源可用状态。
- **添加资源**：手动补充单个资源链接。

## 7. API 接口

所有接口都需要带上系统 API Key：

```http
x-api-key: YOUR_API_KEY
```

### 订阅管理

| 方法 | 路径 | 说明 |
| :--- | :--- | :--- |
| GET | `/api/subscriptions` | 获取订阅列表 |
| GET | `/api/subscriptions/preview` | 预览订阅创建信息 |
| GET | `/api/subscriptions/remote-resources` | 获取远程订阅资源 |
| POST | `/api/subscriptions/task-preview` | 预估自动创建任务结果 |
| POST | `/api/subscriptions` | 创建订阅 |
| PUT | `/api/subscriptions/:id` | 更新订阅 |
| POST | `/api/subscriptions/:id/refresh` | 立即刷新/校验订阅 |
| DELETE | `/api/subscriptions/:id` | 删除订阅 |

### 资源管理

| 方法 | 路径 | 说明 |
| :--- | :--- | :--- |
| GET | `/api/subscriptions/:id/resources` | 获取订阅资源列表 |
| POST | `/api/subscriptions/:id/resources` | 添加订阅资源 |
| DELETE | `/api/subscriptions/resources/:id` | 删除订阅资源 |
| GET | `/api/subscriptions/resources/:id/browse` | 浏览资源目录 |

---

## 使用建议

- 大订阅源建议先使用 **选择订阅链接**，避免一次导入过多无关资源。
- 开启 **自动创建任务** 前先查看任务预估，确认默认账号和保存目录正确。
- 定期使用 **立即校验** 或 **全部校验** 清理失效资源。
