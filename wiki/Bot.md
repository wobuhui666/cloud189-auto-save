# Telegram 机器人

Telegram Bot 可以让您在手机或桌面 Telegram 中查看任务、选择账号、搜索 CloudSaver 资源、创建任务、追剧和查看订阅状态。

---

## 1. 接入配置

进入 **系统** 页，在 **Telegram 机器人** 中配置：

| 配置项 | 说明 |
| :--- | :--- |
| Bot Token | 从 @BotFather 获取 |
| 默认 Chat ID | 默认交互和通知接收对象 |
| 反代 API 域名 (可选) | Telegram API 反代地址 |
| 允许使用的 Chat ID 列表 | 允许控制 Bot 的用户或群组 |
| 管理员 Chat ID 列表 | 可执行管理员操作的 Chat ID |
| 成功通知 | 是否发送成功通知 |
| 失败通知 | 是否发送失败通知 |
| 刮削通知 | 是否发送刮削通知 |

配置完成后可以点击 **测试配置**。

如果服务器无法直接访问 Telegram，请同时配置 **系统** 页中的网络代理。

---

## 2. 基础使用流程

推荐顺序：

1. `/accounts` 选择账号。
2. `/fl` 查看常用目录。
3. 直接发送 `cloud.189.cn` 分享链接创建任务。
4. `/search_cs` 搜索 CloudSaver 资源。
5. `/tasks` 查看任务状态。

如果未选择账号，多数需要账号上下文的命令会提示：`请先使用 /accounts 选择账号`。

## 3. 常用命令

| 命令 | 说明 |
| :--- | :--- |

| `/stats` | 查看系统统计 |

| 命令 | 说明 |
| :--- | :--- |

| 命令 | 说明 |

## 4. 直接发送分享链接

Bot 支持直接接收 `cloud.189.cn` 分享链接。

正常模式下：

1. 发送分享链接。
2. Bot 解析资源名称。
3. Bot 弹出常用目录按钮。
4. 点击目录后创建任务。

如果当前账号没有常用目录，Bot 会提示先添加常用目录。

---

## 5. 静默模式

通过 `/silent` 可以开启或关闭静默模式。

开启后：

- Bot 处理分享链接时不再询问保存目录。
- 会直接使用当前账号的默认常用目录创建任务。

适合高频从 Telegram 转存资源的场景。

---

## 6. CloudSaver 搜索模式

发送 `/search_cs` 后进入搜索模式。

交互规则：

- 输入关键字进行搜索。
- 输入编号选择结果并转存。
- `/cancel` 退出搜索模式。
- 3 分钟无操作自动退出。
- 如果此时直接发送分享链接，Bot 会自动退出搜索模式并改为处理该分享链接。

---

## 7. 自动追剧

Bot 支持：

```text
/series 剧名 年份
/lazy_series 剧名 年份
```

它会使用 **系统** 页中的自动追剧默认账号和默认保存目录，与 Web 端自动追剧逻辑一致。

---

## 8. 权限说明

- 只有 **允许使用的 Chat ID 列表** 中的对象可以控制 Bot。
- 某些危险操作仅管理员可用，例如：
  - `/execute_all`
  - `/retry_<ID>`
  - `/dt_<ID>`
  - `/df_<ID>`

---

## 使用建议

- 初次使用先配置一个默认账号和几个常用目录。
- 高频转存用户建议开启静默模式。
- 搜索不到资源时，优先确认 CloudSaver 配置是否正确。
