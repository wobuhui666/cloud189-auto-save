# Alist 集成

Alist 是一款支持多种存储的目录列表程序。Cloud189 Auto Save 与 Alist 结合，可以用于 STRM 文件生成和播放时的直链换取。

---

## 1. 配置

进入 **媒体** 页，在 **Alist 设置** 中填写：

| 配置项 | 说明 |
| :--- | :--- |
| 启用 | 开关，关闭后 Alist 相关功能不可用 |
| Alist 地址 | Alist 服务地址，如 `http://127.0.0.1:5244` |
| Token | Alist 的访问令牌 |

系统会检查 `alist.enable`、`alist.baseUrl`、`alist.apiKey` 三项都存在时才会启用 Alist 功能。

---

## 2. STRM 文件生成

本项目支持通过 Alist 挂载的目录批量生成 STRM 文件。

需要在 **账号** 页配置账号的：

- **云端媒体目录前缀**：Alist 中访问该文件的基础路径。
- **本地目录前缀**：STRM 生成到本地的目标路径。

---

## 3. 播放直链换取

当 STRM 启用了系统中转时，系统会通过内置的代理机制换取最新直链。如果天翼云接口变动但 Alist 能正常播放，通过 Alist 接口换取直链可作为备选方案。

---

## 4. 任务执行后刷新缓存

当任务转存完成后，如果满足以下条件，系统会自动调用 Alist 接口刷新目录缓存：

- Alist 已启用
- 账号配置了云端媒体目录前缀
- 任务未使用系统代理模式

这可以确保转存完成后，通过 Alist 挂载的目录能及时看到新文件。

---

## 5. 与 STRM 配置的关系

**STRM** 页中的 **普通 (账号/目录)** 类型配置可以配合 Alist 使用。配置中的云端路径会作为 Alist 的目录路径，本地路径作为 STRM 的输出目录。

---

## 使用建议

- 建议将 Alist 与本项目部署在同一局域网或同一宿主机内，减少 API 调用的网络延迟。
- 如果只是使用天翼云盘直连，可以不启用 Alist。
