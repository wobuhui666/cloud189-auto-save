# Telegram 机器人指南

Cloud189 Auto Save 集成了强大的 Telegram 机器人，让您可以随时随地通过手机控制转存任务。

## 1. 机器人配置

### 获取 Bot Token
1. 在 Telegram 中关注 [@BotFather](https://t.me/BotFather)。
2. 发送 `/newbot`，按照提示设置名称 and ID。
3. 获取生成的 `HTTP API Token`。

### 获取 Chat ID
1. 在 Telegram 中关注 [@userinfobot](https://t.me/userinfobot)。
2. 向其发送任意消息，获取您的 `Id`。

### 系统接入
1. 进入 Cloud189 Auto Save 的“系统设置” -> “Telegram 配置”。
2. 填入 `Bot Token` 和 `Chat ID`。
3. 勾选“启用机器人”并保存。

---

## 2. 常用指令列表

### 基础控制
- `/help` - 显示机器人使用帮助。
- `/accounts` - 显示已配置的账号列表，可点击切换当前活跃账号。
- `/tasks` - 显示最近的任务列表，支持分页与快速操作。
- `/cancel` - 取消当前正在进行的交互（如搜索或目录选择）。

### 任务执行
- `/execute_all` - 一键触发所有待处理任务的转存。
- `/execute_[ID]` - 执行指定 ID 的任务。
- `/strm_[ID]` - 为指定任务生成 STRM 文件。
- `/emby_[ID]` - 触发指定任务对应的 Emby 库刷新。
- `/dt_[ID]` - 删除任务（可选是否同步删除云盘文件）。

### 自动化追剧
- `/series [剧名] [年份]` - 创建一个普通转存追剧任务。
- `/lazy_series [剧名] [年份]` - 创建一个“懒转存”追剧任务。
- *注意：追剧功能会使用系统预设的默认账号和默认保存目录。*

## 3. 核心功能操作

### 交互式创建任务
1. **直接发送链接**：将天翼云盘分享链接直接发给机器人。
   - 格式：`https://cloud.189.cn/t/xxxx (访问码: xxxx)`。
2. **选择目录**：机器人会弹出该账号的“常用目录”列表。
3. **确认创建**：点击目录名称，任务即刻创建完成。

### 资源搜索 (`/search_cs`)
1. 发送 `/search_cs` 进入搜索模式。
2. 发送关键词（如“复仇者联盟”）。
3. 机器人会返回 CloudSaver 搜集到的天翼云盘资源列表。
4. 输入资源前的**数字编号**，即可直接启动转存流程。

### 影巢资源搜索 (`/hdhive`)
1. 发送 `/hdhive` 进入影巢搜索模式，或直接使用 `/hdhive 关键字` 搜索。
2. 输入关键词后，机器人会返回影巢/TMDB 匹配结果。
3. 可以直接点击结果按钮，或输入结果前的**数字编号**，机器人会继续查询该条目的天翼资源。
4. 如果资源已解锁或免费，可以直接点击资源按钮，或输入资源编号进入创建任务流程。
5. 如果资源需要积分，机器人会先弹出确认按钮，确认后再调用影巢解锁。

### 影巢签到 (`/hdhive_checkin`)
- 发送 `/hdhive_checkin` 可触发一次影巢签到。
- 该能力依赖 Browser Bridge 签名模式已配置完成。

### 目录管理
- `/fs` - 进入交互式目录树浏览模式，您可以像资源管理器一样点击文件夹，找到目标位置后点击“确认”将其添加为常用目录。
- `/fl` - 查看并管理已添加的常用目录。

---

## 4. 注意事项
- **安全性**：机器人仅响应配置好的 `Chat ID` 发来的消息，其他人无法控制您的系统。
- **代理设置**：如果您的服务器在中国大陆且无法直接访问 Telegram 服务器，请在“系统设置”中配置 `Telegram 代理`。
- **超时**：资源搜索模式在 3 分钟无操作后会自动退出。
