# 功能深度解析

本指南详细介绍 Cloud189 Auto Save 的核心功能模块及其高级用法。

---

## 1. STRM 矩阵 (STRM Matrix)

STRM 文件允许媒体服务器（如 Emby/Jellyfin/Plex）直接播放云端资源，而无需下载到本地。

### 实时 STRM 生成
- **原理**：每当转存任务发现新文件时，系统会自动在映射的 `strm` 目录生成对应的 `.strm` 文件。
- **配置**：需在账号设置中配置 `云端路径前缀`（Alist 中的访问路径）和 `本地路径前缀`。

### 懒转存 (Lazy STRM)
- **场景**：您不想立即转存整个 4K 影集，只想在播放时才触发转存。
- **流程**：
  1. 在“懒转存”页面输入分享链接。
  2. 系统生成“虚拟”的 `.strm` 文件。
  3. 当播放器请求该文件时，系统后台立即执行转存，并实时返回天翼云直链。
- **优势**：极大地节省了您的网盘空间，实现了“即看即转”。

### 播放代理 (Stream Proxy)
- **系统中转**：开启后，`.strm` 中的链接指向本项目地址，由项目在播放时动态换取天翼云最新直链，彻底解决链接过期导致的“无法播放”问题。

## 2. AI 智能辅助

通过集成 OpenAI 或 智谱 AI，系统可以实现更智能的文件管理。

### 智能重命名
- **AI 模式**：发送文件名给 AI，AI 会根据剧集规则自动重命名（例如：将 `[某某字幕组] 某某剧集 S01E05 1080P.mkv` 简化为 `S01E05.mkv`）。
- **配置**：在“系统设置” -> “AI 配置”中填入 `API Key` 和 `Base URL`。

### 自动刮削 (Scraping)
- **TMDB 联动**：系统会自动抓取 TMDB 上的海报、背景图、剧集简介。
- **NFO 生成**：自动生成符合 Emby/Jellyfin 标准的 `.nfo` 元数据文件。

---

## 3. 媒体中心联动 (Emby/Jellyfin)

### 自动通知
- **API 联动**：转存完成后，系统通过 API 通知 Emby 刷新特定文件夹，无需全库扫描。

### 路径替换规则
- **痛点**：天翼云盘路径（如 `/影视/电影`）与 Emby 挂载路径（如 `/media/aliyun/电影`）不一致。
- **解决方案**：在账号设置中配置 `Emby 路径替换`。
  - 格式：`云端路径:本地映射路径`
  - 示例：`/影视:/media/cloud`
  - 效果：系统会将通知中的路径自动转换为 Emby 识别的本地路径。

---

## 4. 高级任务管理

### 追更模式 (Auto Series)
- **智能检测**：系统记录已转存的集数，当分享链接内出现新集数时，自动触发转存。
- **过滤规则**：支持通过正则表达式包含或排除特定后缀（如排除 `.txt`, `.jpg`）。

### 回收站自动清理
- **定时任务**：天翼云盘转存频繁会产生大量回收站碎文件，系统可定时自动清空个人及家庭云回收站，释放云空间。

## 5. 影巢资源 (HDHive)

影巢功能用于把资源发现、天翼分享解析和转存任务串起来，减少手动搜索、复制链接和反复登录的操作。

### 资源搜索与查询
- **关键词搜索**：在 Web 控制台“影巢”页输入关键词，系统会读取影巢公开页面并解析可见资源。
- **TMDB 查询**：对于已匹配 TMDB ID 的电影或剧集，可直接查询对应影巢天翼资源。
- **资源信息展示**：列表会展示清晰度、字幕、大小、发布时间、积分消耗和是否免费等信息。

### Browser Bridge 登录态
- **网页登录取 Cookie**：配置 Browser Bridge 后，可通过常驻 Chromium 完成影巢网页登录，并把 Cookie 同步回主项目。
- **云端状态持久化**：Bridge 可配合 Postgres 保存加密后的 Cookie/StorageState，容器重启后自动恢复登录态。
- **辅助操作**：支持读取当前用户、签到、积分日志、资源详情和已解锁天翼链接。

### 免费资源与积分保护
- **免费资源识别**：系统会识别 `免费` 与 `0 积分` 资源，已解锁资源会优先只读回查详情，避免重复解锁。
- **付费确认**：资源需要积分或积分未知时，前端会弹出确认提示，防止误操作造成积分消耗。
- **详情回读**：解锁后会重新读取资源详情，确认是否拿到天翼分享链接和访问码。

### 一键转存流程
1. 在“影巢”页搜索资源，或从媒体条目按 TMDB ID 查询资源。
2. 选择免费资源或确认解锁付费资源。
3. 系统解析天翼分享链接和访问码。
4. 点击转存后自动带入创建任务流程，可继续生成 STRM、追更和通知媒体库刷新。

### 配置建议
- **公开部署必须设置 `BRIDGE_TOKEN`**：Bridge 敏感接口需要访问令牌保护，避免被外部滥用。
- **推荐配置云数据库**：设置 `DATABASE_URL` 和 `BRIDGE_STATE_SECRET` 后，Bridge 可加密持久化登录态。
- **账号密码放主项目设置页**：不建议把影巢账号密码直接写入 Bridge 环境变量，便于轮换和减少暴露面。

## 6. 资源订阅 (Subscriptions)

订阅功能允许您批量导入并自动监控外部资源集合（如通过 UUID 分发的资源包）。

- **工作流**：
  1. 在“资源订阅”页面输入订阅 UUID。
  2. 系统自动解析该订阅包含的所有分享链接。
  3. 系统定时轮询订阅更新，发现新资源后根据预设规则自动执行转存或生成 STRM。
- **管理**：支持对订阅内的资源进行批量校验、标记失效或一键转存到指定目录。

---

## 7. 定时与自动化 (Automation)

系统的所有自动化行为均由 `SchedulerService` 驱动：
- **Cron 表达式**：支持在“系统设置”中通过标准的 5 段或 6 段 Cron 表达式灵活配置检查频率。
- **并发控制**：内置任务队列，防止多任务同时转存导致的天翼云盘接口限速或并发冲突。
